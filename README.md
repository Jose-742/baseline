# Projeto baseline
## 🚀 Começando
Sistema de cadastro com autenticação por E-mail e redefinição de senha. 
### 🔧 Tecnologias usadas
* Spring boot 3.0.2
    * spring security
    * devtools
    * starter-web
    * jpa
    * validation
    * starter-mail
* thymeleaf 
* bootstrap 5
* JavaScript
* Java 17
* mysql
* jquery
